# 分足デイトレード戦略 設計書

スイング戦略(README参照)の検証で「持ち越しシグナルの1日圧縮はエッジが消える」ことが確認されたため、
日中の値動きに固有のエッジを持つ3手法で構成するデイトレード専用戦略。

## 全体構造

```
前夜: ウォッチリスト選定(日足) → 9:00-9:15 寄付きレンジ観測
→ モジュールA/B/C でエントリー → 場中リスク管理 → 15:20 全決済
```

## ウォッチリスト選定(前夜、日足データ)

- 基礎条件: プライム市場 / 株価300円以上 / 20日平均売買代金10億円以上
- in-play条件(いずれか):
  1. 前日出来高 ≥ 20日平均の1.5倍
  2. 当日が決算翌日(/equities/earnings-calendar で判定)
  3. 寄付きギャップ |±2%| 以上
- スコア = 予想RVOL × |ギャップ率| の上位20銘柄

## モジュールA: ORB(寄付きレンジブレイクアウト)

| 項目 | ルール |
|---|---|
| レンジ | 9:00〜9:15 の高値・安値(OR)。OR幅が株価の2%超の日は見送り |
| エントリー | 9:15〜11:30、5分足終値がOR高値を上抜け。足の出来高が平均分足出来高の2倍以上、価格>VWAP、ギャップ方向と一致 |
| 損切り | ORの中値。リスク1.5%超なら見送り |
| 利確 | +2Rで半分、残りは5分足終値のVWAP割れ |
| 回数 | 1銘柄1日1回 |

## モジュールB: ギャップ&ゴー

| 項目 | ルール |
|---|---|
| 対象 | 決算・材料で寄付き+3%以上ギャップアップ |
| セットアップ | 9:00〜9:05の初足が陽線かつ終値がレンジ上位50% |
| エントリー | 初足高値ブレイク(9:05〜10:00限定) |
| 損切り | 初足安値。リスク2%超なら見送り |
| 利確 | Aと同じ |

## モジュールC: VWAP押し目

| 項目 | ルール |
|---|---|
| トレンド日判定 | 10:00時点で9:15以降の5分足終値の80%以上がVWAP上、かつVWAP上向き |
| エントリー | VWAPタッチ(±0.15%)後の陽線5分足の次足。12:30〜13:00は執行しない |
| 損切り | VWAP -0.5% |
| 利確 | 当日高値更新で半分、終値がVWAP-0.3%割れで全部 |
| 稼働 | 10:00〜15:00、1銘柄1日2回まで |

## リスク管理

- 1トレードのリスク = 資金の0.5%
- 同時2銘柄、1日最大4トレード
- 日次損失 -2% でその日は新規停止(リベンジトレード禁止の機械化)
- 15:20 全ポジション強制決済(持ち越しゼロ)
- コスト前提: 手数料0円、スリッページ片道0.05%

## 実装メモ

- データ: /equities/bars/minute(V2、Premiumプラン限定)。列: Date, Time, Code, O, H, L, C, Vo, Va
- 取得戦略: ウォッチリスト銘柄×該当日のみ取得しキャッシュ(全量取得は数千万行で非現実的)
- 大引け時刻: 2024-11-05 に15:00→15:30へ変更。強制決済は「大引け10分前」を日付で切替
- 5分足に集約してから判定(1分足はノイズとデータ量の問題)
- 同一足内で損切り・利確が両成立した場合は損切り優先(保守的)
- 検証順序: Phase1 = ORB単体・直近1年 → Phase2 = B/C追加 → Phase3 = 期間延長・感度分析
